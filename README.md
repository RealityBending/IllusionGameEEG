# IllusionGameEEG

This dataset was collected in 2022-2023 and comprises electroencephalography, physiological, and behavioural data acquired from 103 healthy individuals (ages: 21-45). The task was administered as part of a larger study.

- [OpenNeuro](https://openneuro.org/datasets/ds004587/versions/1.0.0)
- [OpenNeuro data via GitHub](https://github.com/OpenNeuroDatasets/ds004587)
